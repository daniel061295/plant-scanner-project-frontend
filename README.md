# Plant Scanner Project Frontend

This is the frontend for the Plant Health App, built with Next.js, React, and Tailwind CSS. It features a modern, clean architecture and integrates with a Python backend for plant diagnostics, history tracking, and subscription management.

## Features
- **Plant Scanning:** Upload photos of your plants to get AI-powered diagnostics.
- **Scan History:** Keep track of past plant scans and their health status.
- **Profile & Subscriptions:** Manage your account settings, UI preferences (Dark mode, Language), and view subscription plans.
- **Clean Architecture:** Organized structure separating Domain, Application, and Infrastructure layers.

## Tech Stack
- [Next.js](https://nextjs.org/) (App Router)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Tabler Icons](https://tabler-icons.io/)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
